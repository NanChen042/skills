这项技能指导我们创建独具特色、达到生产级别的界面，避免千篇一律的“AI垃圾”美学。编写真正可用的代码时，要格外注重美学细节和创意选择。

用户提供前端需求：需要构建的组件、页面、应用程序或界面。他们可能还会提供有关用途、受众或技术限制等背景信息。

## 设计思维

在编写代码之前，请先理解上下文，并确定一个大胆的美学方向：

* **目的**：此界面解决了什么问题？哪些用户会使用它？

* **风格**：选择一个极端风格：极简主义、极繁主义、复古未来主义、有机/自然、奢华/精致、趣味/玩具感、杂志/杂志风格、粗犷/原始、装饰艺术/几何、柔和/粉彩、工业/实用主义等等。风格选择众多。您可以从中汲取灵感，但要设计出真正符合美学方向的作品。

* **限制条件**：技术要求（框架、性能、可访问性）。

* **差异化**：是什么让它令人难忘？人们会记住的一件事是什么？

**关键在于**：选择一个清晰的概念方向并精准执行。大胆的极繁主义和精致的极简主义都可行——关键在于意图，而非强度。

然后实现可运行的代码（HTML/CSS/JS、React、Vue 等），即：

* 生产级且功能齐全

* 视觉效果惊艳，令人难忘

* 与清晰的审美观点相契合

* 每个细节都经过精心打磨

## 前端美学指南

重点关注：

* **字体选择**：选择美观、独特且引人注目的字体。避免使用 Arial 和 Inter 等通用字体；而应选择能够提升前端美感的独特字体；选择出人意料、富有个性的字体。将醒目的标题字体与精致的正文字体搭配使用。

* **色彩与主题**：坚持统一的美学风格。使用 CSS 变量来保持一致性。主色调搭配鲜明的点缀色比平淡均匀的配色方案效果更佳。

* **动态效果**：使用动画来实现特效和微交互。优先考虑纯 CSS 的 HTML 解决方案。如果 React 有 Motion 库，请使用它。专注于高冲击力时刻：精心设计的页面加载，配合错落有致的动画延迟，比分散的微交互更能带来惊喜。使用滚动触发和悬停状态来制造惊喜。

* **空间构成**：出人意料的布局。不对称。重叠。对角线流动。打破网格的元素。大量的留白或控制密度。

* **背景与视觉细节**：营造氛围和深度，而非仅仅使用纯色。添加与整体美感相符的上下文效果和纹理。运用渐变网格、噪点纹理、几何图案、分层透明、戏剧性阴影、装饰性边框、自定义光标和颗粒叠加等创意形式。

永远不要使用通用的 AI 生成的美学元素，例如过度使用的字体系列（Inter、Roboto、Arial、系统字体）、陈词滥调的配色方案（尤其是白色背景上的紫色渐变）、可预测的布局和组件模式，以及缺乏上下文特征的千篇一律的设计。

发挥创意，做出出人意料的选择，让设计真正契合语境。每个设计都不应雷同。尝试浅色和深色主题、不同的字体、不同的美学风格。切勿让不同世代的设计都采用相同的风格（例如 Space Grotesk）。

**重要提示**：实现复杂度应与美学理念相匹配。极繁主义设计需要复杂的代码，以及大量的动画和特效。极简主义或精致设计则需要克制、精准，并注重间距、字体和细微之处。优雅源于对理念的完美诠释。
